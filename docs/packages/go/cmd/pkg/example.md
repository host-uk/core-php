# Package Examples

## Search

```bash
core pkg search core-
core pkg search api
core pkg search --org myorg
```

## Install

```bash
core pkg install core-api
core pkg install host-uk/core-api
```

## List

```bash
core pkg list
core pkg list --format json
```

## Update

```bash
core pkg update
core pkg update core-api
```

## Outdated

```bash
core pkg outdated
```
